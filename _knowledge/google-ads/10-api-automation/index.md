# API 与自动化

Google Ads API 是实现广告投放规模化管理的利器。通过 API，你可以自动化账户管理、批量操作、数据抓取和报告生成，大幅提升工作效率。

### [API 概述](api-overview/)
Google Ads API 与 AdWords API 的区别、版本演进、适用场景与核心能力。

### [认证与权限](api-authentication/)
开发者令牌申请、OAuth 2.0 配置、服务账号设置、权限管理最佳实践。

### [常用操作](api-common-operations/)
账户查询、广告系列创建、关键词批量操作、出价调整、报告生成。

### [Python 自动化](python-automation/)
使用 google-ads-python 库构建自动化脚本，实现批量操作和数据同步。

---

## 为什么使用 Google Ads API

| 场景 | 手动操作 | API 自动化 |
|------|----------|-----------|
| 管理 10+ 账户 | 耗时易错 | 一键批量执行 |
| 每日报告生成 | 重复劳动 | 定时自动输出 |
| 关键词批量扩展 | 逐个添加 | 批量导入上万词 |
| 动态出价调整 | 反应滞后 | 实时数据驱动 |
| 跨账户数据汇总 | 导出合并 | 直接聚合分析 |
