## 核心概念速览

| 概念 | 说明 |
|------|------|
| 账户 (Account) | Google Ads 顶级容器，包含所有广告资源 |
| 广告系列 (Campaign) | 围绕特定目标创建的广告集合 |
| 广告组 (Ad Group) | 包含关键词和广告的最小管理单元 |
| 质量得分 | Google 对广告质量的 1-10 分评价 |
| 广告排名 | 决定广告展示位置的计算公式 |
