# 搜索广告基础

搜索广告是 B2B 企业获取高意向潜在客户的核心渠道。当目标客户在 Google 搜索与您业务相关的关键词时，您的广告将精准展示在搜索结果页。

## 一、搜索广告核心概念

### 1.1 什么是搜索广告

搜索广告（Search Ads）是 Google Ads 中最基础、最常用的广告形式。当用户在 Google 搜索框输入关键词时，相关广告会展示在搜索结果页的顶部或底部。

**B2B 搜索广告的优势：**
- 精准触达主动寻找解决方案的潜在客户
- 按点击付费，预算可控
- 效果可追踪，ROI 可量化

### 1.2 搜索广告的基本结构

```
账户 (Account)
├── 广告系列 (Campaign)
│   ├── 广告组 (Ad Group)
│   │   ├── 关键词 (Keywords)
│   │   ├── 广告文案 (Ads)
│   │   └── 落地页 (Landing Page)
```

## 二、关键词策略基础

### 2.1 B2B 关键词特征

与 B2C 不同，B2B 关键词具有以下特点：

| 特征 | B2B | B2C |
|------|-----|-----|
| 搜索量 | 较低但精准 | 较高但泛化 |
| 关键词长度 | 长尾词为主 | 短词为主 |
| 决策周期 | 数周至数月 | 几分钟至几天 |
| CPC | 较高（¥50-200+） | 较低至中等 |

### 2.2 关键词分类

**按搜索意图分类：**
- **信息型**：用户寻求知识（"什么是 CRM"）
- **调研型**：用户比较方案（"CRM 系统对比"）
- **交易型**：用户准备购买（"CRM 系统价格"）

> 建议优先投放交易型和调研型关键词。

## 三、匹配类型概述

Google Ads 提供三种主要匹配类型：

| 匹配类型 | 符号 | 触发范围 | 适用场景 |
|----------|------|----------|----------|
| **精确匹配** | `[keyword]` | 仅精确匹配或紧密变体 | 高价值词，预算有限 |
| **词组匹配** | `"keyword"` | 包含关键词的短语 | 平衡流量与精准度 |
| **广泛匹配** | `keyword` | 相关搜索及同义词 | 探索新关键词，预算充足 |

详细内容请参考 [匹配类型详解](match-types-guide/)。

## 四、广告系列结构设计

### 4.1 B2B 广告系列组织原则

**按产品/服务分类：**
```
广告系列：CRM 软件
├── 广告组：CRM 功能介绍
├── 广告组：CRM 价格咨询
└── 广告组：CRM 实施服务

广告系列：ERP 系统
├── 广告组：ERP 功能模块
└── 广告组：ERP 行业方案
```

**按购买阶段分类：**
```
广告系列：认知阶段（信息型关键词）
广告系列：考虑阶段（调研型关键词）
广告系列：决策阶段（交易型关键词）
```

### 4.2 广告组最佳实践

- 每个广告组包含 5-20 个紧密相关的关键词
- 每个广告组至少 3 个广告变体
- 广告文案与关键词高度相关
- 落地页内容与广告承诺一致

## 五、B2B 线索生成要点

### 5.1 高转化落地页要素

1. **清晰的标题**：直接回应搜索意图
2. **价值主张**：明确说明产品/服务优势
3. **社会证明**：客户案例、评价、认证
4. **明确的 CTA**："免费试用"、"获取报价"、"预约演示"
5. **简化表单**：只收集必要信息

### 5.2 B2B 线索质量优化

- **使用否定关键词**：排除非目标客户（如"免费"、"兼职"）
- **地理位置定位**：聚焦业务覆盖区域
- **受众细分**：利用再营销列表、类似受众
- **投放时段优化**：根据业务时间调整出价

## 六、常见问题 (FAQ)

### Q1: B2B 搜索广告需要多少预算？

**A:** 建议初期日预算不低于 ¥300-500，以确保获得足够的数据进行优化。具体预算应根据关键词 CPC 和目标转化成本确定。

### Q2: 多久能看到效果？

**A:** 通常需要 2-4 周的数据积累期。前两周主要用于收集数据和识别问题，第三周开始优化，第四周评估效果。

### Q3: 搜索广告和展示广告如何选择？

**A:** 建议优先投放搜索广告，因为搜索意图明确，转化率更高。展示广告适合品牌曝光和再营销，可作为补充。

## 总结

搜索广告是 B2B 数字营销的基础，成功的关键在于：

1. **精准的关键词策略** - 聚焦高意图词
2. **合理的账户结构** - 清晰的分类和组织
3. **高质量的广告文案** - 与搜索意图匹配
4. **优化的落地页** - 提升转化率
5. **持续的优化迭代** - 数据分析与调整
