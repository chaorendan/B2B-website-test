# Performance Max

Performance Max 是 Google 的全渠道自动化广告解决方案，利用机器学习在 Google 所有渠道寻找高价值客户。

## PMax 适用场景

- **多渠道投放**：希望在搜索、展示、YouTube、Gmail 等渠道统一投放
- **转化目标明确**：有清晰的转化事件和追踪体系
- **预算充足**：需要一定数据量让机器学习发挥作用
